***AJOUTS***

-L'affichage des informations sur les ressources du joueurs sont afficher dans le haut du cadre de la partie.
-Les joueurs peuvent maintenant sélectionner une planète et passer à la vue planétaire de celle-ci s'ils l'ont colonisé.
-La création des structures est maintenant possible à partir du menu de la vue planète.
-Emplacements de construction de bâtiments maintenant présents sur la planète.
-Les ressources de la planète sont visibles en tout temps dans la vue planète.
-Les joueurs peuvent accéder à chacune des vues : Galaxie, Planete, Solaire, SolaireMere.
-Les joueurs ont un bouton pour les amener directement au système solaire de leur planète mère.
-Les joueurs peuvent maintenant créer des vaisseaux.
-Les joueurs peuvent déplacer leur vaisseaux dans leur système solaire.
-Les joueurs peuvent attaquer, les vaisseaux des autres joueurs.
-La vue initiale des joueurs est le système solaire qui contient leur planète mère.
-Exploration et colonisation possible; l'envoie d'un vaisseau sur une planète inconnu nous l'approprie temporairement. Son appropriation devient permanente si on la colonise et créant une structure sur celle-ci.
-Système de chat en place, fonctionne multidiffusion et monodiffusion.
-Implémentation de l'interface d'arbre de technologies.
-L'interface technologie permet des bonus économiques et des bonus militaires avec la création de vaisseaux.
-Amélioration de l'esthétisme de la vue système solaire et planète.
-Affichage du lobby et de l'écran d'accueil modifié pour un meilleur esthétisme.
-Les systèmes solaires/étoiles ont maintenant un nom.
-Les planètes ont un nom.
-Les ressources sont aléatoirement attribué pour chacune des planètes.
-Les vaisseaux peuvent se déplacer d'un système solaire à la galaxie, puis accéder à un nouveau système solaire par la galaxie.
-Les planètes bougent autour d'un soleil dans chacun des systèmes solaires.
-Une capitale est créer dans la planète-mère à chaque commencement de partie. 

***INSTRUCTIONS***

Pour faire apparaître la vue Galaxie:
	-Cliquer sur le bouton Galaxie.
La vue galaxie sera afficher.

Pour faire apparaître la vue Planete:
	-Cliquer sur votre planète colonisé
	-Cliquer sur le bouton planète
La vue de votre planète apparaîtra.
	
Pour faire apparaître la vue SolaireMere:
	-Cliquer sur le bouton SolaireMere.
La vue SolaireMere sera afficher.	

Pour accéder à un autre système solaire:
	-Déplacer votre vaisseau vers une galaxie, en cliquant sur la galaxie.
	-Cliquer sur "Vers le Soleil" pour permettre au vaisseau d'accéder à ce système solaire.
	-Cliquer sur le bouton Solaire pour accèder à la vue du système solaire.
La vue Solaire prendra place dans ce système solaire.

Pour accéder à son système solaire-mère à partir d'un autre système solaire:
	- Cliquer sur le bouton SolaireMere.
La vue de notre système solaire s'affichera.
	
-Cliquer sur bouton Tech & Vaisseaux pour faire apparaitre arbre de tech.
	- Cliquer sur le bouton Economie:
					- Celui-ci permet la croisée de deux chemins qui procurent différents avantages aux joueurs.
					- Ce sont des bonus qui sont reliées à l'économie du joueurs.
	
	
	- Cliquer sur le bouton Militaire:
					- Celui-ci permet de créer le bouton des Vaisseaux qui ont des avantages différents de chacun des autres.

Pour créer un vaisseau sur la planète-mère:
	- Cliquer sur le bouton Tech & Vaisseaux
	- Cliquer sur l'onglet Militaire
	- Cliquer sur le nom du vaisseau
	- Cliquer sur l'image du vaisseau
Le vaisseau va apparaître.
	
Pour activer un bonus économique:
	- Cliquer sur le bouton Tech & Vaisseaux
	- Cliquer sur le bouton Economie
	- Cliquer sur le bouton du Bonus souhaité
Le bonus économique de votre choix sera appliqué. 
	
Pour attaquer un autre vaisseau:
	- Déplacer votre vaisseau dans le rayon de portée d'attaque pour qu'un attaque soit initialiser.
Le combat est commencé.
	

Pour créer une structure sur une planète:
	- Cliquer sur un planète colonisé de votre couleur.
	- Cliquer sur le bouton Planete.
	- Cliquer sur un des carrés qui représente l'emplacement des structures.
	- Cliquer sur un des boutons qui permet la construction d'une structure.
La structure a été créé.
	
Pour accéder à la vue Galaxie avec un vaisseau:
	- Cliquer sur le vaisseau souhaité.
	- Cliquer sur le bouton "Vers la Galaxie"
Le vaisseau apparaîtra dans la vue Galaxie et disparaîtra de la vue système solaire.

Pour coloniser une planète:
	- Cliquer sur le vaisseau souhaité
	- Cliquer sur la planète à coloniser.
Si la planète est inconnue elle nous appartient temporairement jusqu'à ce qu'on bâtisse une structure sur celle-ci. 
Elle devient alors nôtre de façon permanente.

Pour communiquer avec les autres joueurs:
	Pour communiquer avec Tous:
		- S'il n'est pas déjà sur "Tous", cliquer sur le menu contenant "Tous" ou le nom d'un joueur.
		- Cliquer "Tous".
		- Tapez le message souhaité au clavier, dans la petite fenêtre blanche.
		- Cliquer sur le bouton "Envoyer"
	Le message affichera dans le champs de messagerie et enverra un message à tous.
	
	Pour communiquer avec uniquement un autre joueur:
		- Cliquer sur le menu contant "Tous" ou un nom de joueur.
		- Cliquer sur le nom du joueur.
		- Tapez le message souhaité au clavier, dans la petite fenetre blanche.
		- Cliquer sur le bouton "Envoyer".
	Le message affichera dans le champs de la messagerie et enverra un message au joueur spécifique.
